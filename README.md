Easy Money

A simple web tool to help track and grow personal savings. Enter your monthly income and see suggested allocations for a Stocks & Shares ISA, a Cash ISA, and how much you can spend.

Features
Calculates saving and spending allocations based on your monthly income.
Accessible error messages for invalid inputs.
Modular and responsive design using HTML, CSS, and JavaScript.
Designed for live updates in future versions.

Technologies Used
HTML5 – semantic structure and accessible markup
CSS3 – modular styling with variables, layout, and components
JavaScript (ES6) – simple functions for calculations and dynamic content
